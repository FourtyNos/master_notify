A centralized notification management script for your automations.

## Problem
Managing multiple automation notifications can be challenging when:
- You have numerous automations sending notifications
- Each automation has its own notification action configuration
- You need to update notification settings (e.g., when changing phones)

## Solution
This script provides a single configuration point for all your automation notifications:
- Configure mobile devices in one place
- Update notification settings once to affect all automations
- Simplified maintenance and updates

### Benefits
- Reduced redundancy
- Easier device management
- Time-saving updates
- More maintainable automation setup
